# User Management

## Roles

- User (level 1): standard access
- Moderator (level 5): moderation tools
- Sysop (level 10): full administration

## Common tasks

- View users: `USERS [pattern]`
- Inspect a user: `USERINFO <username>`
- Promote/Demote: `G @user=MODERATOR` or `G @user=10`
- Kick session: `KICK <username>`
