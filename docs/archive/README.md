# Documentation Archive

This directory contains historical development documentation, design proposals, and planning documents that are no longer part of the active documentation set but are preserved for reference.

## Directory Structure

### design/
System design documents, game proposals, and technical specifications from the development process.

- Game designs (Battleship, Blackjack, etc.)
- System architecture proposals (MUD/MUSH, NPC dialogues, currency, etc.)
- Technical research and API investigations

### planning/
Implementation plans, phase documents, and feature specifications used during development.

- Phase implementation plans (PHASE1-PHASE11)
- Feature specifications (rooms, NPCs, tutorials, etc.)
- Migration checklists and implementation guides

### milestones/
Development milestone documentation and completion tracking.

- Integration completion markers
- Testing setup documentation
- Development standards

### content/
NPC dialogue content and game data (not user-facing documentation).

## Note

These documents represent the design and planning process but are not maintained as current documentation. For up-to-date information, see the main documentation directories:

- `/docs/user-guide/` - Player documentation
- `/docs/administration/` - Admin and sysop guides
- `/docs/getting-started/` - Installation and setup
- `/docs/development/` - Current development documentation
